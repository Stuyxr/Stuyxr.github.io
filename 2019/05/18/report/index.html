findMedianSortedArrays

题目含义：给两个有序数组，寻找中位数。

算法思想：

中位数具有划分作用：中位数将数组划分为两个子集，一个子集中的元素 $\leq$ 另一个子集的元素。

保证划分left_part中元素个数与right_part中元素个数相等，或者右边比左边多以1个，那么最后中位数$x = \frac{max(left_part) + min(right_part)}{2}$。

分析得知：

$i, j$ 应保证一下两个条件:

  1. $i + j = \frac{m + n + 1}{2}$

  2. $B[j - 1]\leq A[i]$ && $A[i - 1]\leq B[i]$
